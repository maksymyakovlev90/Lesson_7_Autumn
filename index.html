СSS властивість Background. Фон для сайту

За допомогою властивості background можна встановити колір, положення, зображення, прив'язку та повторюваність бекграунду як окремого елемента, так і всього сайту. Останнє, по суті, є завданням для тега body. Розглянемо всі властивості, пов'язані із фоном (background). Якщо ви тільки навчаєтеся, то найкращим варіантом засвоєння уроку буде практичне застосування властивості в такому порядку, як я розглядаю нижче:

Background-color

Встановлює колір фону. Можна застосовувати до окремих елементів h1, p або до всього сайту через body:

/* чорний фон сайту */
body {
background-color: #000;
}


/* чорний фон заголовка, білий колір шрифту */
h1 {
color: #fff;
background-color: #000;
}

Background-image

Використовується для вставки фонового зображення, при цьому в css вказуємо посилання на картинку:

body {
background-color: #000;
background-image: url ("my-image.jpg");
}

Зверніть увагу як вказаний шлях до зображення - це означає, що воно має знаходитися в тій же папці, що і css файл стилів. А якщо ні, то слід вказувати повний шлях до зображення. Опція стане в нагоді якщо ви захочете, наприклад, додати клікабельний фон на сайті.

Background-size

Визначає розмір фонового зображення.

div {
background: url(my-image.jpg);
background-size: 100% 100%;
background-repeat: no-repeat;
}

Для властивості є кілька цікавих особливостей. Крім числових значень розмірів (px,pt) можна використовувати відсоткові або автоматичне обчислення (auto - наприклад, для однієї зі сторони картинки - 500px auto). Ця властивість застосовується для створення фону для сайту з картинкою на весь екран.

Якщо вказати значення cover, то фон масштабуватиметься щоб ширина і висота картинки = ширині та висоті блоку. Якщо використовувати значення contain, масштабування буде зі збереженням пропорцій щоб зображення повністю помістилося в блок.

Background-repeat

Зображення, що використовується в попередньому прикладі, буде «розтиражовано» (повторюється) по всьому екрану. Ця властивість покликана керувати цим процесом. Є кілька допустимих значень:

  • background-repeat: repeat-x — зображення повторюється лише по горизонталі
  • background-repeat: repeat-y — зображення повторюється лише по вертикалі
  • background-repeat: repeat — зображення повторюється по горизонталі та вертикалі
  • background-repeat: no-repeat — зображення не повторюється

background-attachment — дана властивість визначає, чи буде фонове зображення фіксуватися при прокручуванні сторінки:

  • background-attachment: scroll — прокручується разом зі сторінкою
  • background-attachment: fixed — при прокручуванні фон залишається нерухомим

Background-position

Задає розташування зображення щодо екрана, за замовчуванням воно виводиться у верхньому лівому кутку. Значення цієї властивості є набором координат Х (по горизонталі) і Y (по вертикалі), які починаються з лівого верхнього кута. Може задаватися:

  • у фіксованих одиницях (пікселі, сантиметри)
  • в процентах
  • словесно: - top (зверху), bottom (знизу), center (по центру), left (ліворуч) і right (праворуч).

Розглянемо приклади:

  • background-position: 20px 50px — зображення відступає вниз на 20 пікселів зверху та праворуч на 50 від лівого краю.
  • background-position: 50% 25% — розташоване по центру по горизонталі та відступає на 25% зверху.
  • background-position: right bottom — зображення розташовується знизу праворуч.

Всі ці настройки можуть бути записані в одній властивості background, порядок дотримання властивостей:

[background-color] | [background-image] | [background-repeat] | [background-attachment] | [background-position]

Наприклад:

background: #000 url("my-image.jpg") no-repeat fixed left bottom;

Якщо якась властивість пропускається, його значення встановлено за замовчуванням.
